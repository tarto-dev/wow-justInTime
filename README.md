# JustInTime

Lightweight raid timing helper for WoW Retail

**Author**: Claralicious_
**Interface**: WoW Retail (Midnight, 12.0)

## Install

Clone the `master` branch directly into `Interface/AddOns/JustInTime/`:

```
cd <wow>/Interface/AddOns
git clone -b master <repo-url> JustInTime
```

Or download the latest release zip and extract.

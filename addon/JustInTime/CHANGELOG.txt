JustInTime — Changelog
=======================

v0.3.0 — Cross-level fallback + death-adjusted timer (2026-04-27)
------------------------------------------------------------------

Features:
- Cross-level reference fallback: when reference data is missing for the
  exact key level, the addon now picks the nearest available level within
  ±7 (e.g. running a +15 NPX with only +18-20 reference data falls back to
  +18). The overlay clearly labels the fallback as "Réf : 28:34 (clé +18)"
  and the chat prints a one-time info line at the start of the run. Beyond
  the ±7 cap the comparison would be too biased, so it falls through to
  "ref unavailable" rather than misleading you.
- Dynamic overlay height: the frame now resizes to fit the actual number
  of bosses in the dungeon (3-boss dungeons get a tighter frame, 4-boss
  unchanged, no wasted space at the bottom).

Fixes:
- Death penalty is now included in the elapsed timer. The addon's "Écoulé"
  matches Blizzard's in-game timer, and your boss splits are comparable to
  reference data which is sourced from death-adjusted leaderboard times.
  Penalty is read from `C_ChallengeMode.GetDeathCount()` so it adapts to
  affixes that scale the cost (e.g. Xal'atath's Guile = 15s/death on +12+).

v0.2.0 — Initial public release (2026-04-25)
---------------------------------------------

First public release of JustInTime on CurseForge.

Features:
- Live pace tracking: continuous delta against a reference run for the same
  dungeon, key level, and affix combo.
- Per-boss table: boss name, your time (live counter on the next-up boss,
  color-graded green / violet / red based on pace), and the reference split.
- Pace bar with progress fill, projected-finish marker, and per-boss pips
  that turn green on kill.
- Reference modes: public (curated reference set), fastest, recent, and
  median of your personal runs. Optional "Ignore affixes" to widen the
  sample when affix-specific data is sparse.
- Visibility modes: always visible during keys, or popup transient (6s
  fade after each boss kill).
- Show overlay button in settings to position the frame anywhere on your
  UI without being in a key.
- Movable overlay with lock / unlock and reset-to-default.
- Chat triggers (toggleable): boss kill, key-end recap, threshold cross,
  deplete-imminent warning.
- Critical alerts: red pulse, chat warning, sound.
- Slash commands: /jit, /jit show, /jit hide, /jit lock, /jit unlock,
  /jit mode <public|fastest|recent|median>, /jit reset, /jit test,
  /jit help.
- Locales: French (default for frFR) and English (enUS / enGB).
- Pure Lua, no LibStub, no Ace3. Single overlay frame, ~10 Hz updates,
  active only when needed.

Compatibility:
- WoW Retail Midnight (Interface 120001).
